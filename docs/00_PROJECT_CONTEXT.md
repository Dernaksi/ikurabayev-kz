# Project Context

## Project

IKurabayev.kz is the repository for the personal research website, QR landing
page, engineering portfolio, and AI-assisted project laboratory for Iskander
Kurabayev.

## Purpose

The repository should provide a durable, reviewable source of truth for public
identity, research portfolio material, engineering work, and the workflow used to
develop the website with AI assistance.

## Primary Audiences

- Research collaborators and reviewers.
- Engineering collaborators and hiring reviewers.
- Conference, event, or QR landing page visitors.
- Iskander Kurabayev and AI agents maintaining the repository.

## Repository Roles

- Public website source: future implementation for `ikurabayev.kz`.
- Content source: reviewed public biography, research, portfolio, and contact
  routing material.
- AI Lab: prompts, workflow, decision records, and repeatable task templates.
- Governance record: rules for what may and may not be published.

## Non-Goals For The Initial Foundation

- No website implementation yet.
- No frontend framework selection yet.
- No package manager setup yet.
- No dependency installation.
- No private data collection.
- No unpublished research data storage.

## Initial Success Criteria

- The repository explains what it is for.
- AI agents have clear operating rules.
- Future tasks can start from issues and pull requests.
- Public content boundaries are explicit before content is added.
